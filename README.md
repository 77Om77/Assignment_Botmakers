# Assignment Botmakers - Full Stack Project

## Project Description
This is a full-stack web application built using React (frontend) and Spring Boot (backend). It includes authentication and role-based access control.

---

##  Tech Stack

### Frontend
- React
- TypeScript
- Vite

### Backend
- Spring Boot
- Java
- Maven

---

##  Project Structure

frontend/ → React application  
backend/ → Spring Boot application  

---

##  How to Run Project

###  Frontend

cd frontend  
npm install  
npm run dev  

---

###  Backend

cd backend  
mvn spring-boot:run  

---

##  Features

- User Registration & Login
- JWT Authentication
- Role-Based Access Control (RBAC)
- Protected Routes
- REST APIs
